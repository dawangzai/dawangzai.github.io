爱摄影上线了

发表于 2017-10-24   |   分类于 wangzai

我独立开发设计的一款 APP “爱摄影” 今天终于上线了！

做 Android 开发也有一段时间了一直想写一款自己的 App 总结一下这段时间的工作积累，但是苦于没有合适的开放 Api ，计划就一直被搁置着，直到年前在朋友圈看到一位学长上线了他 App “尚图” 从而知道了 Unsplash 网站。 打开一看我靠它居然提供了一套完整的 Api 接口，有一种发现新大陆的赶脚，现在米已有就等下锅啦。

从开发到放弃

既然有了 Api 那就开干吧。但是当时还是刚入门不久的菜鸟，虽然很认真的在做但是还是没弄出什么结果来。封装完整的网络框架，搭建合适的 App 架构等对于当时的我来说还是太难了，没有经验和知识积累又想处处都做得合理到最后只能是处处都不合理，捣鼓了个把月只弄出个半成品，只能暂时搁置了。

爱摄影上线了

这段时间又重新拾起了这个项目，并取名 “爱摄影” 。目标是集成 Unsplash Api 提供的所有功能，把它做成一个有意义的东西，虽然不一定有什么用 ~~

目前 “爱摄影” 只有浏览照片以及用户登录这两块，虽然只有一小部分但也算是前进了一大步（哈哈）。
